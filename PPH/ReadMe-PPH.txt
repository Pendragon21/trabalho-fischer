Gerador dos Coefficientes a0, a1, ..., aN e b0, b1, ..., bN.


Entrada:  n - grau dos polinômios.
	  n_inst - número de instâncias a gerar com os parâmetros acima

Formato do Arquivo:

O arquivo possui 3+n números inteiros. Todos os separadores
são brancos exceto pelos CR e LF no final das linhas.

O primeiro inteiro contém  "n".  
Os n+1 inteiros seguintes correspondem aos coeficientes a0, a1, ..., aN.
Os n+1 inteiros restantes correspondem aos coeficientes b0, b1, ..., bN.

Se você preferir, pode copiar o gerador para o seu código
e utilizálo sem gerar arquivos. Faça isso com atenção.
